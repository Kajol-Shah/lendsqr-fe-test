# Lendsqr Frontend Engineering Test

This project is a submission for the **Frontend Engineer Assessment** by Lendsqr.

It replicates a simplified version of the Lendsqr Admin Dashboard, built using **React + TypeScript + SCSS**, following the provided Figma design.

👉 https://lendsqr-fe-test-five-dun.vercel.app/

## 📁 GitHub Repository

## 📌 Project Overview

The application includes:

- Login Page (UI Only)
- Dashboard Layout
- Users Page (with mock API data)
- User Details Page
- Fully responsive UI based on Figma design

---

## ⚙️ Tech Stack

- React (Vite)
- TypeScript
- SCSS
- React Router DOM
- React Icons
- Mock API (local JSON / mock service)
